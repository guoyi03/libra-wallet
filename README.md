# Libra Wallet


### Environments
* Python 3.7
* PyQT5
* Ubuntu
* [Libra](https://github.com/libra/libra)
* [libra explorer](https://libexplorer.com/)

### Install
```
#conda create -n wallet python=3.7
#conda activate wallet
(wallet) #pip install -r requirements.txt
```
### Run
```
(wallet) #python wallet.py
```
